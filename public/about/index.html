远洋渔船保障系统管理后台（全栈开发） 链接到标题

时间：2025-09 ~ 2025-11
角色：全栈开发

项目概述：团队承接的远洋渔船保障人员训练管理后台，覆盖训练看板、人员管理、学习资源管理等核心模块，支持教练制定训练计划与学习资源布置。

个人职责与成果：

  1. 负责学习资源管理模块，设计并实现网格式目录管理页面与颜色标记交互，提升资源分类与查找效率。
  2. 实现文件批量上传、上传进度展示、大文件分片上传与断点续传，显著增强大体积资料上传稳定性。
  3. 基于 ECharts 封装可复用可视化组件，并集成多种文档格式预览能力，提升训练数据与资源展示效果。
  4. 使用 Pinia 实现角色权限控制，结合路由守卫完成权限验证与登录状态检查，保障系统访问安全。
  5. 引入 CDN 加速静态资源，优化首屏加载性能与页面响应速度。

技术栈：Vue 3 + ECharts + Axios + Pinia + Spring Boot + MyBatis + MySQL + Redis

项目代码地址： https://github.com/wangqiuZGL/Git_ChuanBo_Optimize

中国铁塔篡改检测系统（前端开发/算法服务协作） 链接到标题

时间：2025-02 ~ 2025-08
角色：前端开发 / 算法服务协作

项目概述：受中国铁塔委托开发的智能检测系统，用于支付凭证、房产证等关键业务图片及 PDF 文件的真实性审核。系统集成图片篡改检测、OCR 信息提取、相似度匹配与报告导出能力。

个人职责与成果：

  1. 负责电表图片篡改检测模块全流程落地，参与 YOLOv11 模型训练与 Python 算法服务开发。
  2. 前端开发电表篡改任务管理页面，实现文件批量上传与 Excel 导出，提升任务处理效率和使用体验。
  3. 通过 keep-alive 保持组件切换状态，并对关键操作按钮增加防抖处理，降低重复提交风险。
  4. 基于 CSS 变量实现主题切换方案，提升界面一致性与可维护性。
  5. 负责数据库与接口实现，并参与项目在铁塔公司服务器的部署落地。

技术栈：Vue 3 + Element Plus + Axios + Pinia + Python + YOLOv11
